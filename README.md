## 업데이트 태그 기준
- **Major:** 새로운 틀의 생성이나 주요 기능 추가
- **Minor:** 세부 기능 추가
- **Patch:** 버그 수정, 주석, README 등 문서 수정
- **Test:** 큰 변화를 위해 개발중. 프로그램이 거의 실행이 안되는 수준. 테스트 브랜치에서 별도 작업(이후 병합되어 표시됨).
- **MainTag-TEST:** 변화를 부분적으로 적용. 작동하지 않는 일부 핵심기능 존재. 메인 브랜치에서 이용.
